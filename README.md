# 🚗 Обновленный АвтоБазар для Arizona RP.

![Lua](https://img.shields.io/badge/Language-Lua-blue.svg)
![Platform](https://img.shields.io/badge/Platform-SAMP-lightgrey.svg)
![Version](https://img.shields.io/badge/Version-3.0-success.svg)

Современная модификация для комфортной торговли транспортом на АвтоБазаре. Скрипт полностью заменяет устаревшие серверные диалоги на плавный интерфейс **mimgui** и добавляет стильные **CEF-уведомления** прямо на экран игрока.

  <img src="https://i.imgur.com/ImXujCz.jpeg" alt="AutoBazaar Banner" width="800">
</p>

---

## ✨ Главные возможности

- **Интерфейс mimgui:** Удобное меню с поиском, фильтрацией по ценам и классам автомобилей.
- **CEF-уведомления:** Всплывающие уведомления о покупке, продаже и предложениях в стиле современных проектов. Не засоряют игровой чат.
- **Аналитика:** Отслеживание статистики побед в аукционах и сливов автомобилей в гос.
- **Оптимизация:** Скрипт потребляет минимум ресурсов и не вызывает просадок FPS даже при большом скоплении машин на АБ.
- **Умный парсинг:** Быстрое и точное считывание цен напрямую с табличек.

---

## 🛠️ Требования

Для корректной работы скрипта в вашей сборке GTA San Andreas должны быть установлены:

* [MoonLoader 0.26](https://blast.hk/threads/13305/)
* Библиотека **mimgui**
* Библиотека **SAMP.Lua**
* Плагин для поддержки CEF в SAMP

---

## 🚀 Установка

1. Скачайте последний релиз скрипта из раздела [Releases](../../releases).
2. Распакуйте содержимое архива.
3. Переместите файл `new_ab.lua` (и папку с ресурсами) в папку `moonloader`, которая находится в корневой директории вашей игры.
4. Запустите игру.

---

## ⌨️ Использование

* `/ab` — открыть главное меню настроек и управления.
* *Скрипт активируется и начинает парсинг автоматически при входе в зону АвтоБазара.*

## 📝 Список изменений (Changelog)

### Версия 3.0
- **Новый интерфейс:** Реализована новая система красивых CEF-уведомлений для вывода информации на экран.
- **Оптимизация:** Оптимизирован парсинг цен с табличек АвтоБазара.
- **Статистика:** Добавлено отслеживание статистики побед в аукционах и сливов автомобилей в гос.
- **Стабильность:** Мелкие исправления багов и повышение общей стабильности работы скрипта.

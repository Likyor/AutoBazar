# Политика безопасности (Security Policy)

Спасибо, что помогаете делать «Обновленный АвтоБазар» лучше и безопаснее! 

## Поддерживаемые версии

Обновления безопасности и исправления критических ошибок выпускаются только для актуальных версий скрипта. Пожалуйста, убедитесь, что вы используете последнюю версию, прежде чем сообщать о проблеме.

| Версия | Поддерживается |
| ------ | ------------------ |
| 3.x  | ✅ Да               |
| 2.x  | ❌ Нет              |
| 1.x  | ❌ Нет              |

## Сообщение об уязвимости

Если вы обнаружили серьезную ошибку или уязвимость (например: краш игры у других игроков, утечку памяти, критический баг в работе CEF-уведомлений или возможность выполнения произвольного кода), пожалуйста, **не создавайте публичный Issue**.

Вместо этого сообщите о проблеме приватно, используя один из следующих способов:

1. **BlastHack:** Напишите мне в личные сообщения на форуме [BlastHack](https://www.blast.hk/members/597654/).
2. **Telegram:** Свяжитесь со мной напрямую: [@cvvdk](https://t.me/cvvdk).
3. **GitHub:** Используйте приватные сообщения об уязвимостях (Private vulnerability reporting) во вкладке **Security** данного репозитория.

### Что следует указать в отчете:

Для того чтобы я мог максимально быстро исправить проблему, пожалуйста, предоставьте следующую информацию:
* Подробное описание уязвимости или критического бага.
* Пошаговую инструкцию для воспроизведения (как именно вызвать проблему на АвтоБазаре).
* Вашу версию MoonLoader, mimgui и CEF-плагина.
* Обязательно прикрепите файл `moonloader.log` (и логи краша, если они есть).

Я стараюсь проверять сообщения ежедневно и отвечу вам при первой возможности. Как только проблема будет подтверждена, я выпущу хотфикс для актуальной версии. 

Спасибо за вашу помощь и внимательность!
